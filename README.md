To-Do List Web Application
This project is a simple and responsive to-do list web application that allows users to manage their tasks effectively.

Features
    Add Tasks: Users can add new tasks via an input field and button.
    Mark as Completed: Clicking a task toggles its "completed" status.
    Delete Tasks: Tasks can be removed individually by clicking the "Ã—" icon.
    Persistent Storage: Tasks are saved in the browser's local storage and persist after a page refresh.
    Responsive Design: The application is mobile-friendly and visually appealing.
  
  
Technologies Used
     HTML: Provides the structure of the application.
     CSS: Styles the application, including layout and animations.
     JavaScript: Implements functionality such as adding, deleting, and storing tasks.
